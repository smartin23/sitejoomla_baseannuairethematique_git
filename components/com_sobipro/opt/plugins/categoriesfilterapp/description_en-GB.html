Categories Filter Addon allows to show a Category Filter to the search results list. It uses a "sid_list" url parameter to enforce a list of categories IDs. Example:
http://demo.prieco.com/index.php?option=com_sobipro&task=search.results&sid=1&sid_list=2,4

NOTE: It requires the Categories Restriction Addon to apply the filter in the results list.

Category List Mode - End Level: number of category levels to be shown.

Categories Order: categories order to be shown.

Category List Mode - Start Level: start showing from this level to the end level.

Category List Restriction: to restrict the categories in the results to this sid_list, no input control is going to be shown, a hidden value "sid_list" will contain these values. Please, note in this list are included all categories beyond the current section (there's no easy eay to filter the categories in the backend). So, select only categories from the current section.
